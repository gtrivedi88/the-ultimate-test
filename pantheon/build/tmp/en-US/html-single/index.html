Preface

Write a short introductory paragraph that provides an overview of the module. The introduction should include what the module will help the user do and why it will be beneficial to the user. Include key words that relate to the module to maximize search engine optimization.

Prerequisites

  • A bulleted list of conditions that must be satisfied before the user starts the steps in this module.
  • Prerequisites can be full sentences or sentence fragments; however, prerequisite list items must be parallel.

Procedure

  1. Make each step an instruction.

  2. Include one command or action for each step with the exception of simple follow-step, for example:

    1. Do this thing and then select Next.
    2. Do this other thing and then select Next.
  3. Use an unnumbered bullet (*) if the procedure includes only one step.

Verification

  1. Provide an example of expected command output or a pop-up window that the user receives when the procedure is successful.
  2. List actions for the user to complete, such as entering a command, to determine the success or failure of the procedure.
  3. Make each step an instruction.
  4. Include one command or action per step.
  5. Use an unnumbered bullet (*) if the verification includes only one step.

Additional resources

  • A bulleted list of links to other closely-related material. These links can include link: and xref: macros.
